## The Scenario

A high-speed projectile (50 m/s) is tracked by a laser targeting gimbal. At `t=0.30s`, optical backscatter introduces **2.5m sensor noise** (plasma scintillation zone) for 800ms.

| Condition | Outcome |
| :--- | :--- |
| **ZTP Disabled** | Sensor noise drives gimbal into jitter. Dwell energy: 758/1000. **Target escaped.** |
| **ZTP Enabled** | Firewall detects innovation anomaly at `t=0.30s`. R covariance dilated in <10µs. Kalman switches to constant-velocity kinematic projection. Dwell energy: 1000/1000. **Target neutralized at t=0.999s.** |

## Physics Model

The Rust kernel runs a **Kalman filter tracker** with ZTP anomaly detection:

```
1. Kinematics    — advance true target: y += vy × dt
2. KF predict    — forecast state covariance
3. ZTP guard     — detrended rolling variance check
                   if |innovation| > 4σ: dilate R to 200.0 (ignore sensor)
4. KF update     — compute Kalman gains, update estimate
5. Gimbal PD     — Kp=1000, Kd=60, saturate at ±5000 rad/s²
```

The ZTP guard prevents **covariance latch-up** — the failure mode where inflated sensor noise fools the Kalman filter into treating measurement noise as signal, causing the gimbal to chase phantom positions.

## Schema

Each row is one **1.0ms control cycle**.

| Column | Description |
| :--- | :--- |
| `condition` | `unprotected` or `ztp_protected` |
| `t` | Elapsed time (s) |
| `y_t` | True target position (m) |
| `y_meas` | Raw optical sensor measurement (m) — noisy during scintillation |
| `y_est` | Kalman filter estimated position (m) |
| `y_g` | Gimbal servo actual position (m) |
| `dist` | Beam offset from true target (m) |
| `dwell_energy` | Cumulative on-target dwell energy (units) |
| `destroyed` | Whether neutralization threshold was reached |
| `anomaly_detected` | ZTP covariance anomaly flag active |

## Loading

```python
from datasets import load_dataset
import pandas as pd

dataset = load_dataset("spiderpilot89/directed-energy-gimbal-1000hz", split="train")
df = dataset.to_pandas()

# Compare beam drift between conditions
protected = df[df["condition"] == "ztp_protected"]
unprotected = df[df["condition"] == "unprotected"]

scintillation = protected[(protected["t"] >= 0.3) & (protected["t"] < 1.1)]
print("Mean beam drift (ZTP):", scintillation["dist"].mean())

scintillation_u = unprotected[(unprotected["t"] >= 0.3) & (unprotected["t"] < 1.1)]
print("Mean beam drift (no ZTP):", scintillation_u["dist"].mean())
```
